五、您依個資法享有的權利

依《個人資料保護法》第 3 條，您對本 App 持有的個人資料享有以下權利：

  • ✅ 查詢或請求閱覽：App 內即時呈現您的所有資料
  • ✅ 請求製給複製本：設定 → 資料備份，可匯出完整 JSON
  • ✅ 請求補充或更正：App 內直接編輯各類資料
  • ✅ 請求停止蒐集、處理或利用：刪除帳號即停止（App 內「刪除帳號」功能）
  • ✅ 請求刪除：App 內「刪除帳號」將同時清除本機資料及 Firestore 雲端資料

六、OCR 功能專項告知

當您使用「匯入對帳單」或「拍照匯入」功能時：

  • 您上傳的圖片或截圖將透過 Google Gemini API 進行文字辨識
  • 圖片內容可能包含您的財務資訊（股票代號、金額、交易紀錄等）
  • Google Gemini API 的資料處理規則適用 Google 隱私權政策
  • 辨識完成後，原始圖片不儲存於本服務資料庫，僅辨識結果（純文字數值）存入您的帳號
  • 首次使用 OCR 功能前，App 將另行取得您的同意

七、拒絕提供的後果

  • 拒絕提供電子郵件（拒絕 Google Sign-In）：無法使用雲端同步功能，可使用純本機模式（如有）
  • 拒絕提供財務資料：App 核心功能（資產追蹤、報告生成）將無法運作，但不影響帳號登入
  • 拒絕 OCR 同意：無法使用 AI 輔助匯入功能，可改用手動輸入或 CSV 匯入
  • 拒絕推播通知：不影響 App 使用，僅無法收到月報生成通知
